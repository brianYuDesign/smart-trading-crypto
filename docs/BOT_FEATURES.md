# 🤖 Smart Crypto Trading Bot - 功能文檔

## ✅ 已實現的功能

### 📰 新聞資訊
查詢最新的加密貨幣新聞，保持市場資訊即時更新。

**指令:**
- `/news [數量]` - 獲取最新新聞（預設 5 則，最多 20 則）
  - 範例: `/news` 或 `/news 10`
- `/latest` - 快速查看最新 5 則新聞

**功能特色:**
- 📡 多來源新聞聚合
- 🕐 顯示發布時間
- 🔗 提供原文連結
- 📱 格式化排版，易於閱讀

---

### 📊 市場數據查詢
即時掌握加密貨幣價格與市場動態。

**指令:**

1. **單一幣種查詢**
   - `/price <幣種>` - 查詢特定幣種的詳細價格資訊
   - 範例: `/price BTC`, `/price ETH`, `/price SOL`
   - 顯示內容:
     - 💵 當前價格
     - 📈 24小時漲跌幅
     - 💎 市值
     - 📊 24小時成交量

2. **主流幣種總覽**
   - `/prices` - 一次查看多個主流幣種（BTC, ETH, BNB, SOL, XRP, ADA）
   - 快速掃描市場主要幣種的價格動向

3. **市場總覽**
   - `/market` - 查看整體加密貨幣市場狀況
   - 顯示內容:
     - 🌐 總市值
     - 📊 24小時總成交量
     - ₿ BTC 市佔率
     - ⟠ ETH 市佔率
     - 🪙 活躍幣種數量
     - 😱 恐懼與貪婪指數

4. **市值排行榜**
   - `/top [數量]` - 查看市值排行（預設 10 名，最多 50 名）
   - 範例: `/top` 或 `/top 20`
   - 包含排名、價格、漲跌幅、市值

5. **熱門幣種**
   - `/trending` - 查看當前最熱門的加密貨幣
   - 顯示搜尋量最高的幣種排行

**數據來源:**
- CoinGecko API (即時市場數據)
- 支援主流幣種: BTC, ETH, BNB, SOL, XRP, ADA, DOGE, DOT, MATIC, AVAX 等

### 📈 技術分析
深入分析價格走勢與技術指標。

**指令:**

1. **價格走勢圖**
   - `/chart <幣種>` - 生成價格走勢圖表
   - 範例: `/chart BTC`
   - 功能: 📊 開發中（即將推出 K 線圖、移動平均線）

2. **技術指標分析**
   - `/analysis <幣種>` - 技術指標分析
   - 範例: `/analysis ETH`
   - 顯示內容:
     - 💵 當前價格
     - 📊 24小時價格區間（最高/最低）
     - 📈 漲跌幅
     - ✅ 趨勢判斷（強勢上漲、溫和上漲、小幅下跌、明顯下跌）

**即將推出:**
- 📊 K 線圖表（多種時間週期）
- 📈 移動平均線（MA7, MA30, MA200）
- 📉 技術指標（RSI, MACD, 布林通道）
- 📊 成交量分析

### ⚙️ 個人訂閱管理
自訂價格提醒，不錯過重要時機。

**指令:**

1. **訂閱提醒**
   - `/subscribe <幣種> [條件]` - 訂閱價格提醒
   - 範例:
     - `/subscribe BTC` - 訂閱 BTC（即時更新）
     - `/subscribe ETH >3000` - ETH 超過 $3000 時提醒
     - `/subscribe SOL <100` - SOL 低於 $100 時提醒

2. **取消訂閱**
   - `/unsubscribe <幣種>` - 取消特定幣種的訂閱
   - 範例: `/unsubscribe BTC`

3. **查看訂閱**
   - `/mysubs` - 查看所有訂閱清單
   - 顯示每個訂閱的幣種和觸發條件

**功能特色:**
- ⏰ 自動監控價格變化
- 🔔 達到條件時即時推送
- 📋 管理多個訂閱
- ⚙️ 自訂觸發條件（開發中）

## 📊 技術架構

### 核心模組
```
src/
├── telegram_commands.py      # 指令處理核心（已升級到 V3）
├── market_data.py             # 市場數據 API
├── news_monitor.py            # 新聞監控
├── market_analyzer.py         # 市場分析
├── signal_generator.py        # 交易信號
└── notifier.py                # 通知系統
```

### API 整合
- **CoinGecko API** - 市場數據與價格查詢
- **CryptoPanic API** - 新聞聚合（多來源）
- **Telegram Bot API** - 訊息推送與指令處理

## 🚀 使用範例

### 情境 1: 早晨市場檢查
```
/latest          # 查看最新新聞
/market          # 查看市場總覽
/prices          # 查看主流幣種價格
```

### 情境 2: 深入研究特定幣種
```
/price BTC       # 查看 BTC 詳細資訊
/analysis BTC    # 技術分析
/chart BTC       # 查看走勢圖（即將推出）
```

### 情境 3: 設定自動提醒
```
/subscribe BTC >95000    # BTC 突破 $95000 時提醒
/subscribe ETH <2800     # ETH 跌破 $2800 時提醒
/mysubs                  # 查看所有訂閱
```

### 情境 4: 追蹤熱門趨勢
```
/trending        # 查看熱門幣種
/top 20          # 市值 Top 20
/news 10         # 最新 10 則新聞
```

## 💡 使用建議

### 新手入門
1. 從 `/start` 開始，了解基本功能
2. 使用 `/help` 查看完整指令清單
3. 試試 `/price BTC` 和 `/market` 熟悉市場數據
4. 訂閱感興趣的幣種 `/subscribe BTC`

### 進階使用
1. 結合多個指令進行全面分析
2. 設定多個訂閱追蹤不同幣種
3. 定期查看 `/trending` 發現新機會
4. 使用 `/analysis` 輔助交易決策

### 最佳實踐
- 📅 每天早晨查看 `/latest` 和 `/market`
- 🎯 設定關鍵價位的訂閱提醒
- 📊 結合新聞與技術分析做決策
- ⚠️ 不要僅依賴 Bot，始終 DYOR（Do Your Own Research）
